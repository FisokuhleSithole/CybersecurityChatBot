#  Cybersecurity Awareness Chatbot

A console-based chatbot for educating South African citizens about cybersecurity threats, including phishing, password safety, and safe browsing habits.

##  Project Overview

This chatbot was developed as part of a cybersecurity awareness campaign to help citizens recognize and avoid common online threats. The bot provides interactive education on various cybersecurity topics in a conversational manner.

##  Features

| Feature | Description |
|---------|-------------|
|  **Voice Greeting** | Plays a recorded WAV greeting when the application starts |
|  **ASCII Logo** | Displays a professional ASCII art logo with colored text |
|  **Typewriter Effect** | Text appears one character at a time for conversational feel |
|  **Colored Console UI** | Different colors for bot responses, user input, and errors |
|  **Interactive Chatbot** | Responds to cybersecurity questions naturally |
|  **Multiple Topics** | Covers 10+ cybersecurity topics |
|  **Input Validation** | Handles empty inputs and unknown questions gracefully |
|  **CI/CD Pipeline** | GitHub Actions automatically builds and tests every commit |

 Cybersecurity Topics Covered

| Topic | What Users Learn |
| **Password Safety** | Creating strong, unique passwords |
| **Phishing Scams** | Recognizing fake emails and messages |
| **Safe Browsing** | Identifying secure websites (HTTPS) |
| **Suspicious Links** | How to spot dangerous URLs |
| **Social Engineering** | Manipulation tactics used by attackers |
| **Two-Factor Authentication (2FA)** | Extra security layer for accounts |
| **Data Breaches** | What to do if your data is stolen |
| **Malware Protection** | Avoiding malicious software |
| **Wi-Fi Safety** | Secure practices for public networks |
